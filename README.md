 Simulador de Sistema de Votação

Simulação de um sistema de votação eleitoral utilizando CoppeliaSim.


 Descrição
O projeto simula o funcionamento de uma urna eletrônica e o processo de votação em ambiente virtual.

 Como executar
1. Abrir o CoppeliaSim
2. Carregar o arquivo da cena
3. Executar a simulação
